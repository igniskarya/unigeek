# UniGeek Firmware v1.2.1

## Bug Fixes

> **T-Display 16MB Battery Fix**
> Fixed incorrect battery ADC pin reference that could cause wrong battery readings on T-Display 16MB

## Supported Boards
- M5StickC Plus 1.1
- M5StickC Plus 2
- LilyGO T-Lora Pager
- M5Stack Cardputer
- M5Stack Cardputer ADV
- LilyGO T-Display 16MB
