# UniGeek Firmware v1.3.0

## New Features

- **NFC (MFRC522)** — MIFARE Classic reader with UID scan, sector auth, memory dump, dictionary attack, static nested attack, and darkside attack
- **GPS & Wardriving** — Live GPS view, WiFi + BLE wardriving with Wigle CSV export, NimBLE on all boards
- **Wigle Integration** — Upload wardrive logs, view stats, manage token (from GPS module and Network menu)
- **Barcode Generator** — Code 128 barcode from typed text or file, with inverted color toggle
- **BLE Detector** — BitChat app user detection
- **Pin Configuration** — Configure I2C and GPS pins from Settings

## Bug Fixes

- Power saving respects blocking action overlays
- ScrollListView flicker, Cardputer keyboard, I2C, T-Lora Pager, and web file manager fixes
- Partition size increased

## Supported Devices

M5StickC Plus 1.1, M5StickC Plus 2, T-Lora Pager, T-Display 16MB, DIY Smoochie, Cardputer, Cardputer ADV
