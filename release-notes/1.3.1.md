# UniGeek Firmware v1.3.1

## What's New

**GPS Scan Mode**
> Choose what to scan during wardriving: WiFi + BLE (default), WiFi Only, or BLE Only. Status bar adapts to show only relevant counts.

**GPS Wardrive Mode**
> Two WiFi scanning methods: Driving (default) uses active scanning for better coverage at speed. Walking uses passive promiscuous sniffing with channel hopping.

**File Viewer**
> Tap any file in File Manager to view its contents. Scrollable text display with scrollbar, page jump with LEFT/RIGHT, line scroll with UP/DOWN.

## Bug Fixes

- **Keyboard Input** — Nav keys (`;` `.` `,` `/`) no longer consumed during text input, fixing number mode decimal input and allowing all characters to be typed
- **Wardrive Upload** — File list now sorted newest-first for easier access to recent logs

## Supported Devices

- M5StickC Plus 1.1
- M5StickC Plus 2
- LilyGO T-Lora Pager
- M5Stack Cardputer
- M5Stack Cardputer ADV
- LilyGO T-Display 16MB

## Download

Pre-built binaries: https://github.com/lshaf/unigeek/releases

Copy the `sdcard/` folder to your SD card root, or use the built-in **WiFi > Network > Download** menu to fetch sample files directly on the device.
